Початковий вигляд сторінки

Візьмемо розмічений HTML-файл і застосуємо до нього різні можливості, що надаються каскадними таблицями стилів. Ось так виглядає наша початкова сторінка.

Каскадні таблиці стилів (CSS)

Каскадні таблиці стилів (CSS) -- сучасний елемент Web-сторінок, що дає Вам широкі можливості форматування сайтів. За допомогою CSS розділяється вміст сторінки і її оформлення. Специфікація CSS1 прийнята консорціумом World Wide Web Consortium.

World Wide Web Consortium (W3C)

W3C заснований в жовтні 1994 року для координації розвитку World Wide Web і повної реалізації її потенціалу шляхом розробки універсальних протоколів, сприяючих її еволюції і гарантуючих її цілісність. Консорціумом керує директор -- Tim Berners-Lee, розробник системи World Wide . W3C підтримується організаціями-членами, він незалежний від фірм-виробників.

Підтримка браузерами

Каскадні таблиці стилів підтримуються браузерами, починаючи з Internet 3 і Netscape Navigator 4. Проте, ні один з них не забезпечує повної підтримки всіх специфікацій. Проте, найбільш важливі властивості коректно обробляються всіма сучасними браузерами.

Стилі "за замовчуванням"

У кожному браузері задано, як він показуватиме текст, розмічений тими або іншими тегами HTML. Більшість браузерів показує теги <H1> і <STRONG> однаково. На ці стилі орієнтуються web-дизайнери при розмітці сторінок. HTML дозволяє також указувати стиль в атрибутах тегів, як наприклад <FONT> і <BODY>. Проблема полягає у тому, що, для того щоб поміняти ці стилі, доведеться виправляти атрибут в кожному тезі окремо.

Новий метод

Каскадні таблиці стилів надають користувачам набагато зручніший метод роботи. Оформлення сторінок тепер не прив'язано до їх вмісту, а знаходиться в окремому файлі. Тепер користувач може вказати браузеру, як повинні виглядати всі теги, наприклад, <H1>.

Практичне застосування CSS

Для використання CSS слід зробити наступне:

  1. Вказати, для яких елементів настроюється формат.
  2. Вказати характеристики вибраних елементів.
  3. Перевірити результат в різних браузерах.

CSS дозволяє змінювати наступні характеристики:

  • Кегль (розмір шрифта)
  • Гарнітуру шрифту (не рекомендується)
  • Колір тексту
  • Вирівнювання (по лівому краю, правому краю, по центру, повне) 4
  • Відступ
  • багато характеристик таблиць і т.д.
